# 🚗 Parking System (Java)

A console-based Parking Lot Management System built using Java.

## 🔧 Features
- Vehicle parking with slot selection
- Time-based billing system
- Entry & exit time tracking
- UPI and Cash payment simulation
- QR code for UPI payment

## 🛠 Technologies Used
- Java
- OOP Concepts
- Java Time API

## ▶️ How to Run
1. Compile the program:
   javac ParkingSystem.java

2. Run:
   java ParkingSystem

## 💡 Note
UPI payment is simulated using a QR code image.
